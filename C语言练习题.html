C语言练习题

2022/1/19
一、选择题

  1. 以下选项中不合法的用户标识符是( )。
    A、abc.c
    B、file
    C、Main
    D、PRINT
    答案：A
  2. 下列标识符组中，合法的用户标识符为( )。
    A、_0123 与 ssiped
    B、del-word 与 signed
    C、list 与*jer
    D、keep%与 wind
    答案：A
  3. C 语言程序是由( )组成。
    A、程序
    B、语句
    C、字符
    D、函数
    答案：D
  4. 以下不是关键字的是( )。
    A、continue
    B、char
    C、If
    D、default
    答案：C
  5. 以下叙述错误的是( )。
    A、一个 C 程序总是从 main 函数开始执行的;
    B、C 程序书写格式自由,一行内可以写几个语句,一个语句可以分写在多行上;
    C、每个语句和数据定义的最后必须有一个分号;
    D、C 语言本身有输入输出语句
    答案：D
  6. 以下叙述中正确的是( )。
    A、C 语言的源程序不必通过编译就可以直接运行
    B、C 语言中的每条可执行语句最终都都将被转换成二进制的机器指令
    C、C 源程序经编译形成的二进制代码可以直接运行
    D、C 语言中的函数不可以单独进行
    答案：B
  7. 下列叙述中错误的是( )。
    A、计算机不能直接执行用 C 语言编写的源程序
    B、C 程序经 C 编译程序编译后，生成后缀为.obj 的文件
    C、后缀为.obj 的文件，经连接程序后生成后缀为.exe 文件
    D、后缀为.obj 和.exe 的文件都可以直接运行
    答案：D
  8. 在 C 语言中，要求参加运算的数必须是整数的运算符是( )。
    A、%
    B、/
    C、!
    D、*
    答案：A
  9. 若变量 c 为 char 型，能正确判断出 c 为小写字母的表达式是( )。
    A、’a’<=c<=’z’
    B、(c>=’a’)||(c<=’z’)
    C、(‘a’<=c)and(‘z’>=c)
    D、(c>=’a’)&&(c<=’z’)
    答案：D
  10. 下面选项中表示合法的数据是( )。
    A、0668
    B、2e4.5
    C、123a
    D、160.
    答案：D
  11. 以下选项中,不合法常量的是( )。
    A、1.234e04
    B、1.234e0.4
    C、1.234e+4
    D、1.234e0
    答案：B
  12. C 语言中最简单的数据类型包括( )。
    A、整型、实型、逻辑型
    B、整型、实型、字符型
    C、整型、字符型、逻辑型
    D、整型、实型、逻辑型、字符型
    答案：B
  13. 能正确表示逻辑关系:“10≥=a≥=0”的 C 语言表达式是( )。
    A、10>=a>=0
    B、a>=0 and a<=10
    C、a>=0||a<=10
    D、a>=0&&a<=10
    答案：D
  14. 设 x 和 y 均为 int 型变量,则以下语句:x+=y;y=x-y;x-=y;的功能是( )。
    A、把 x 和 y 按从大到小排列
    B、把 x 和 y 按从小到大排列
    C、无确定结果
    D、交换 x 和 y 中的值
    答案：D
  15. C 语言中, 下列字符常量合法的是( )。
    A、’\t’
    B、”A”
    C、65
    D、A
    答案：A

二、编程题

  1. 输入三个整数作为三角形的三条边，判断如果不能构成一个三角形，则输出“not a triangle”;否则如能构成一个直角三角形，则输出”yes”;否则输出”no”。 复习：C 库函数 - scanf()
  1. 请编程实现从键盘输入10个数，求输入的正数之和。 复习:C 数组
  1. 请编程实现输出自然数1到1000中含有数字2的数的个数。 复习:C语言计数器、C语言如何提取个,十,百,千位上的数字
