一、选择题

  1. 以下选项中不合法的用户标识符是( )。
    A、abc.c
    B、file
    C、Main
    D、PRINT
    答案：A
  2. 下列标识符组中，合法的用户标识符为( )。
    A、_0123 与 ssiped
    B、del-word 与 signed
    C、list 与*jer
    D、keep%与 wind
    答案：A
  3. C 语言程序是由( )组成。
    A、程序
    B、语句
    C、字符
    D、函数
    答案：D
  4. 以下不是关键字的是( )。
    A、continue
    B、char
    C、If
    D、default
    答案：C
  5. 以下叙述错误的是( )。
    A、一个 C 程序总是从 main 函数开始执行的;
    B、C 程序书写格式自由,一行内可以写几个语句,一个语句可以分写在多行上;
    C、每个语句和数据定义的最后必须有一个分号;
    D、C 语言本身有输入输出语句
    答案：D
  6. 以下叙述中正确的是( )。
    A、C 语言的源程序不必通过编译就可以直接运行
    B、C 语言中的每条可执行语句最终都都将被转换成二进制的机器指令
    C、C 源程序经编译形成的二进制代码可以直接运行
    D、C 语言中的函数不可以单独进行
    答案：B
  7. 下列叙述中错误的是( )。
    A、计算机不能直接执行用 C 语言编写的源程序
    B、C 程序经 C 编译程序编译后，生成后缀为.obj 的文件
    C、后缀为.obj 的文件，经连接程序后生成后缀为.exe 文件
    D、后缀为.obj 和.exe 的文件都可以直接运行
    答案：D
  8. 在 C 语言中，要求参加运算的数必须是整数的运算符是( )。
    A、%
    B、/
    C、!
    D、*
    答案：A
  9. 若变量 c 为 char 型，能正确判断出 c 为小写字母的表达式是( )。
    A、’a’<=c<=’z’
    B、(c>=’a’)||(c<=’z’)
    C、(‘a’<=c)and(‘z’>=c)
    D、(c>=’a’)&&(c<=’z’)
    答案：D
  10. 下面选项中表示合法的数据是( )。
    A、0668
    B、2e4.5
    C、123a
    D、160.
    答案：D
    复习:c语言中怎样用e表示指数？
  11. 以下选项中,不合法常量的是( )。
    A、1.234e04
    B、1.234e0.4
    C、1.234e+4
    D、1.234e0
    答案：B
  12. C 语言中最简单的数据类型包括( )。
    A、整型、实型、逻辑型
    B、整型、实型、字符型
    C、整型、字符型、逻辑型
    D、整型、实型、逻辑型、字符型
    答案：B
  13. 能正确表示逻辑关系:“10≥=a≥=0”的 C 语言表达式是( )。
    A、10>=a>=0
    B、a>=0 and a<=10
    C、a>=0||a<=10
    D、a>=0&&a<=10
    答案：D
  14. 设 x 和 y 均为 int 型变量,则以下语句:x+=y;y=x-y;x-=y;的功能是( )。
    A、把 x 和 y 按从大到小排列
    B、把 x 和 y 按从小到大排列
    C、无确定结果
    D、交换 x 和 y 中的值
    答案：D
  15. C 语言中, 下列字符常量合法的是( )。
    A、’\t’
    B、”A”
    C、65
    D、A
    答案：A
  16. 下列关于赋值运算符的表述中,错误的是( )。
    A、赋值运算符优先级高于逗号运算符
    B、赋值运算符的结合性是从左至右
    C、赋值运算符是双目运算符
    D、赋值运算符可在表达式中连用
    答案：B（？）
  17. 设 a 和 b 均为 double 型变量,且 a=2.5、b=3.5,则表达式(int)a+b/b 的值是
    ( )。
    A、3.5
    B、3.500000
    C、3
    D、3.000000
    答案：D
    C 强制类型转换
  18. 在以下运算符中，优先级最高的运算符是( )。
    A、<=
    B、/
    C、!=
    D、&&
    答案：B
  19. 条件运算符的结合方向是( )。
    A、自右至左
    B、自左至右
    C、任意
    D、前后顺序
    答案：A
    复习:C运算符
  20. 设 char ch=’A’;则 ch=(ch>=’A’&&ch<=’Z’)?(ch+32):ch 的值是( )。
    A、A
    B、a
    C、Z
    D、z
    答案：B
    ASCII码一览表，ASCII码对照表
  21. 若有以下程序段:int c1=1,c2=2,c3;c3=1.0/c2*c1;则执行后,c3 中的值是( )。
    A、0
    B、0.5
    C、1
    D、2
    答案：A
  22. 执行程序段 int a=-1,b=4,k; k=(++a<0)&&!(b–<=0);
    printf(“%d %d %d\n”,k,a,b);后的输出结果是( )。
    A、1 0 4
    B、1 0 3
    C、0 0 3
    D、0 0 4
    答案：D
    解析
  23. 若变量 c 为 char 型，能正确判断出 c 为小写字母的表达式是( )。
    A、’a’<=c<=’z’
    B、(c>=’a’)||(c<=’z’)
    C、(‘a’<=c)and(‘z’>=c)
    D、(c>=’a’)&&(c<=’z’)
    答案：D
  24. 设 int x=1, y=1; 表达式(!x||y–)的值是( )。
    A、0
    B、1
    C、2
    D、-1
    答案：B
    解析
  25. 以下关于 long、int 和 short 类型数据占用内存大小的叙述中正确的是 ( )。
    A、均 占 4 个字节
    B、根据数据的大小来决定所占内存的字节数
    C、由用户自己定义
    D、由 C 语言编译系统决定
    答案：D
    复习:各数据类型存储的字节数
  26. 下面程序的输出结果是( )。
    #include <stdio.h>
    int main( )
    { int x=7,y=3;
    printf(“%d”,y=x/y);
    return 0;
    }
    A、0
    B、2
    C、3
    D、不确定的值
    答案：B
  27. 以下叙述错误的是( )。
    A、C 语言区分大小写
    B、C 程序中的一个变量，代表内存中一个相应的内存单元，变量的值可以根据
    需要随时修改
    C、整数和实数都能用 C 语言准确无误地表示出来
    D、在 C 程序中，正整数可以用十进制、八进制和十六进制的形式来表示
    答案：C
  28. 若有定义：int x=3,y=2;float a=2.5,b=3.5；则 (x+y)%2+(int)a/(int)b
    的值为( )。
    A、1.0
    B、1
    C、2.0
    D、2
    答案：B
  29. 已知 char a; int b; float c; double d; 则表达式 a*b+c-d 结果为( )型。
    A、double
    B、int
    C、float
    D、char
    答案：A
    解析
  30. 以下程序的输出结果是( )。
    #include <stdio.h>
    int main( )
    {
    int y=3,x=3,z=1;
    printf(“%d %d\n”,(++x,y++),z+2);
    return 0;
    }
    A、3 4
    B、4 2
    C、4 3
    D、3 3
    答案：D
    解析
  31. 已知下列语句中的 x 和 y 都是 int 型变量，其中错误的语句是( )。
    A、x=y++ ;
    B、x=++y ;
    C、(x+y)++ ;
    D、x+y++
    答案：D
  32. 以下程序的输出结果是( )。

    B、4
    C、6

  33. 已知 int a=6;则执行 a+=a-=a*a;语句后，a 的值为( )。
    A、36
    B、0
    C、-24
    D、-60
    答案：D
  34. sizeof(char)是（ ）。
    A) 一种函数调用
    B) 一个整型表达式
    C) 一个字符型表达式
    D) 一个不合法的表达式

    A) 8,10,10
    B) 10,10,10
    C) 8,8,10
    D) 8,16,10

  36. 以下选项中，合法的一组 C 语言数值常量是( )。

    B、 12 OXa23 4.5e0

    A、 ‘'‘
    B、 ‘\t’
    C、 ‘\n’
    D、 ‘088’
    答案：D
  38. 若有以下定义和语句：
    int a=010,b=0x10,c=10;

    A、8,10,10

  39. 以下说法正确的是( )。
    A、输入项可以为一个实型常量，如 scanf(“%f”,3.5)；
    B、只有格式控制，没有输入项，也能进行正确输入，如 scanf(“a=%d,b=%d”);
    C、当输入一个实型数据时，格式控制部分应规定小数点后的位数，如
    scanf(“%4.2f”,&f);
    D、当输入数据时，必须指明变量的地址，如 scanf(“%f”,&f);
    答案：D

    }

二、编程题

  1. 输入三个整数作为三角形的三条边，判断如果不能构成一个三角形，则输出“not a triangle”;否则如能构成一个直角三角形，则输出”yes”;否则输出”no”。 复习：C 库函数 - scanf()
  1. 请编程实现从键盘输入10个数，求输入的正数之和。 复习:C 数组

  1. 请编程实现输出自然数1到1000中含有数字2的数的个数。

    复习:C语言计数器、C语言如何提取个,十,百,千位上的数字

  2. 请编程实现输出1到100以内能被3或5整除，但不能同时被3和5整除的数及其个数。

  3. 请编程实现求解sum=1+2+3+…..+n<s的最大的sum值，其中s的值由键盘输入。

  4. 请编程实现求解s=1/(12)+1/(23)+1/(34)+……+1/(1011)，输出结果保留两位小数。

  5. 请编程实现如下问题的求解。猴子吃桃问题：猴子第一天摘下若干个桃子，当即吃了一半，还不过瘾，又多吃了一个。第二天早上又将第一天剩下的桃子吃掉一半，有多吃了一个。以后每天早上都吃了前一天剩下的一半零一个。到第 10 天早上想再吃时，发现只剩下一个桃子了。编写程序求猴子第一天摘了多少个桃子。

  6. 有一分数序列，2/1, 3/2, 5/3, 8/5, 13/8, 21/13,……，请编程求出这个数列的前20项之和。

  7. 请编程实现以下功能：读入两个运算数（data1和data2）及一个运算符（op），计算表达式data1 op data2的值，其中op可为+，—，*，/（用switch语句实现）。
